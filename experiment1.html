Part 1: Read the following instructions for the game:


  1. Read the instruction and try to understand it "wiku shau netse"
  2. Click and drag the blocks to solve the given instruction. The yellow floor marks the spot where you can place the block. Additionally, you can remove any block from the task.
  3. Press submit button. If the task is correct press next. If the task is incorrect try again!
  4. You have a number of attempts on each task shown in the left corner.
  5. You can add blocks, remove blocks, and place blocks over other blocks.

About the game:


You have entered blockworld, inhabited by people whose only interest is in arranging colored blocks into patterns.

Your task is to learn the blockworld language, use it to arrange blocks, and finally create new instructions in the langauge. It’s a difficult task, but we think you’re up to it!

Phase 1

Your goal is to learn the blockworld language, and to follow instructions in the language to arrange blocks.

You will see several tasks. Each task includes a instructions in the blocksworld language, a starting pattern of blocks that you need to modify, and a workspace that you can take blocks from or put them into. You will have a few chances to find the right pattern. If you run out of chances, you'll be shown the right one and move to the next task.

Phase 2

You will use what you have learned to create instructions for another person. You will see the starting pattern, the pattern the person must create, and the blocks in the person's workspace. You will have a special keyboard to write instructions in the blocksworld langauge, using the sounds the language is built out of.

Another person might have to use your instructions, so choose them carefully!

There are several words to learn and it might be difficult to remember everything. Just do your best and know that it's ok to make mistakes.

Warning: Please don't use the back button to try to go back to an earlier task. Just do the best you can on each task.
